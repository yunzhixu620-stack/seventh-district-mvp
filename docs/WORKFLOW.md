# Delivery Workflow

This MVP is produced in one delivery cycle while preserving traceable staged commits: benchmark and design, technical foundation, structured content, engine, UI and audio, quality assurance, then delivery.

Validation before release includes type checking, unit tests, production build, end-to-end browser testing, responsive visual review, and a full online-session pass after GitHub Pages deployment.

The initial version is deliberately local-only: no external AI requests, accounts, multiplayer synchronization, or server persistence.
